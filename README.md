## üî• Main Features

- Signup
- Login
  - with email & password
  - oauth login
- messaging:
  - Reply
  - Edit
  - Forward
  - Mention
  - Delete
  - Pin
  - Search
  - block/unblock users
- Groups and channels
  - Adding/removing members
  - chanage permissons
  - make others admins
  - Deleting group/channel
  - Edit channel privacy settings
- Voice Calls
- Admin Dashboard
- Add/Remove/view/delete Stories
- Edit profile settings
- Session Management
